# 集いの広場

## サイト概要

同じ趣味同士の人がつながれる、参加のしやすい場所

### サイトテーマ

グループ・人とのつながり・気楽に

### テーマを選んだ理由

私は多趣味で、色々な人と趣味について語り合いたいと考えていて、同じような人が集まれる場所を作りたかったからです

### ターゲットユーザ

老若男女問わず

### 主な利用シーン

趣味について、話したいときいつでも気楽に
